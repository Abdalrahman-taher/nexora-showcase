# Nexora

> Commercial Multi-Tenant ISP SaaS Platform for Internet Service Providers

## Overview

Nexora is a commercial-grade ISP management platform designed for Internet Service Providers (ISPs). It combines business operations, customer management, billing, MikroTik automation, FreeRADIUS integration, monitoring, and network management into a single multi-tenant SaaS solution.

> **Note:** This repository is a project showcase only. The source code is private because it is a commercial product under active development.

---

## Key Features

- Multi-tenant SaaS architecture
- Platform Owner, Company Staff, and Customer portals
- Customer management
- Internet plans & subscriptions
- Billing and invoices
- Wallet & auto-renew
- Payment review workflow
- MikroTik RouterOS integration
- FreeRADIUS REST authorization
- Dynamic bandwidth policies
- Quota & enforcement engine
- Network monitoring
- Topology discovery
- Collector agents
- Support ticket system
- Inventory management
- IP Pool management
- Audit logging
- Telegram notifications
- Reports & analytics
